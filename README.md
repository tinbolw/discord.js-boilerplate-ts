# A barebones [discord.js](https://discord.js.org/) bot boilerplate, in TypeScript. 

The majority of the code is taken verbatim from the discord.js [official guide](https://discordjs.guide/). The only meaningful change is that the deploy scripts rely on a `.env` file instead of a `config.json` file. The only dependency is [discord.js](https://www.npmjs.com/package/discord.js). There are also numerous tweaks applied to ensure compatibility with TypeScript.

For further details, read the [official guide](https://discordjs.guide/).

This project is not in any way officially connected or affiliated with Discord or discord.js.
